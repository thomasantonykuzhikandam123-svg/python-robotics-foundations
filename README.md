# Python Robotics Foundations

OOP concepts applied to robotics using Python — part of my self-learning journey 
towards becoming a Robotics Engineer.

## What this covers
- Classes and objects — Robot and Sensor classes
- Composition — Robot holding Sensor objects
- Class variables, classmethods, staticmethods
- Inheritance — MobileRobot, CleaningRobot, ArmRobot
- Decorators, dunder methods, @property
- NumPy — sensor data, positions, distance calculations, rotation matrix
- SensorDataLogger — real-time sensor data analysis
- RobotFleet — managing multiple robots

## Files
- robotoop.py — main robot system
- sensor_logger.py — NumPy based sensor data logger
- numpy_practice.py — NumPy fundamentals and robotics maths

## How to run
python3 robotoop.py
